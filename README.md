### EV Battery Health Analysis & Predictive Maintenance

**Course Project — Data Science & Analysis (PH 6130), IIT Hyderabad**

A data-driven machine learning framework for analyzing lithium-ion battery degradation using the NASA Ames PCoE Battery Aging Dataset.

This project implements an end-to-end pipeline covering:

* Feature engineering from discharge cycles
* State of Health (SoH) estimation
* Remaining Useful Life (RUL) prediction
* Exponential degradation modeling
* Rule-based predictive maintenance system *(Healthy / Monitor / Replace)*

**Models Compared:** Ridge Regression, Random Forest, XGBoost
**Dataset Scale:** 2,794 discharge cycles across multiple Li-ion battery cells
